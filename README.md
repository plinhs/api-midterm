This project is a cloud-based Mobile Billing System that exposes versioned REST APIs for Mobile Providers, Banking Apps, Website Payments, and Admin functionalities.
The system includes an API Gateway, rate limiting, authentication support, Swagger documentation, and deployment on cloud services.

The system allows:

*Mobile app users to query their bills
*Banking apps to fetch unpaid bills
*Website users to pay bills
*Administrators to create bills (single or batch)
*The project includes:
*API versioning (/api/v1)
*Swagger documentation
*Authentication-ready structure (JWT optional)
*SQL database storage
*API Gateway (Azure APIM)
*Rate limiting
*Request logging
*Cloud deployment
